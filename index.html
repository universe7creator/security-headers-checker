Frequently Asked Questions

What are HTTP security headers?

Security headers are HTTP response headers that help protect web applications from common attacks like XSS, clickjacking, and data injection.

Which headers are most important?

HSTS, Content-Security-Policy, X-Frame-Options, and X-Content-Type-Options are considered essential for basic security.

How often should I scan my website?

We recommend scanning after any deployment or configuration change, and at least monthly for production sites.
