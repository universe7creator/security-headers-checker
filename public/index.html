Frequently Asked Questions

What security headers does it check?
The tool analyzes 15+ headers including Strict-Transport-Security (HSTS), Content-Security-Policy (CSP), X-Frame-Options, X-Content-Type-Options, X-XSS-Protection, Referrer-Policy, Permissions-Policy, Cross-Origin-Embedder-Policy, Cross-Origin-Opener-Policy, Cross-Origin-Resource-Policy, and more.
How is the security grade calculated?
Grades are based on header coverage (which headers are present), configuration quality (proper values), and best practices. A = Excellent (12+ headers), B = Good (9-11), C = Fair (6-8), D = Poor (3-5), F = Critical (0-2).
Can I scan any website?
Yes, you can scan any publicly accessible website. The tool makes HTTP requests to analyze response headers. Some sites with strict bot protection may block scans.
Is there a limit on scans?
No limits! Once you purchase, you get unlimited security header scans. Use it for your own sites, client sites, or security research.
Do I get updates?
Yes! All purchases include lifetime updates. As new security headers emerge and standards evolve, the tool will be updated to check them.
